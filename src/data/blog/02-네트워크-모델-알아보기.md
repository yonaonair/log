# TCP/IP와 OSI 7계층

- 네트워크 모델 : 수많은 네트워크를 통일된 형태로 연결할 방법

## TCP/IP 모델

- 인터넷이 생기기전 1960년 후반부터 비국 국방성 ARPA에서 만들기 시작 
- 1980년, 인터넷 원형인 아르파넷 ARPAnet의 표준으로 지정
- 당시 컴퓨터는 연구소나 대학에서 사용 → 아르파넷은 연구 목적이었음 
- 이후 컴퓨터 대중화에 따라 인터넷에 그대로 사용

## OSI 7계층 모델

- 네트워크가 발전하고 기능이 다양해짐 → 각 계층을 좀 더 세분화할 표준 필요 
- 1984년 ISO(국제 표준화기구)에서 제안한 네트워크 모델

# 네트워크 패킷

- 다양한 계층으로 나뉜 프로토콜을 이용해서 데이터를 전달할 때는, 필요한 내용을 각각 용도에 맞는 프로토콜과 조합해야함 → 조합한 데이터와 프로토콜의 집합 **“패킷”**
