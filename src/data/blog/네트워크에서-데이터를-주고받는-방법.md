#  (1) 대상 지정

데이터를 보낼 때는 대상 컴퓨터를 지정해야 함 

### 유니캐스트 `unicast` 

- 데이터를 "특정 컴퓨터 1대"에만 보냄 
- 가장 일반적 

### 멀티캐스트 `multicast` 

- 같은 네트워크에 있는 "특정 컴퓨터 여러대"에 보냄 
- 특수한 경우에만 사용 

### 브로드캐스트 `broadcast`

- 같은 네트워크에 있는 "모든 컴퓨터"에 보냄
- 특정 대상을 지정할 수 없을 때 사용 

'→ 이후 주소 체계로 연결' 

## (2) 프로토콜 `protocol`

- 네트워크에서 데이터를 주고받을 때 사용하는 양식 

### 1. Ethernet 

- 이더넷
- 가까운 곳의 컴퓨터와 

### 2. IP 

- internet protocol
- 멀리 있는 컴퓨터와 

### 3. TCP/UDP

- transmission control protocol / user datagram protocol 
- 특정 컴퓨터에서 실행중인 프로그램에 데이터 전달
